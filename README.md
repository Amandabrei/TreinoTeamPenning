Aplicação criada para ajudar em um esporte que meus filhos praticam chamado Team Penning, 
primeiro criei um cadastro dos nomes, clicando no botão "Gerar Times" forma trios na tabela Classificatória, onde a primeira coluna tem os trios sorteados, a segunda coluna colocamos manualmente o número do boi que é sorteado em uma fotocélula externa na hora do treino,
e a terceira coluna colocamos o tempo que cada trio executou o Team Penning, um botão para finalizar a Classificatória, onde é lançado apenas os 15 menores tempos para correr uma final, na tabela final primeira coluna os trios classificados, 
segunda coluna o tempo feito na classificatória, terceira coluna local para colocar o número do boi, quarta coluna tempo da final, um botão Selecionar Campeões, onde faz a somatória do tempo da Classificatória e Tempo da Final, 5 menores tempos ganham.

Aplicação feita usando HTML, JavaScript e CSS (ainda não finalizei).
